PEP8

  • * Usar 4 espacios para identar, no mezclar tabulaciones y espacios.
  • * Máximo de 79 caracteres por linea.
  • * Usar parentesis para encerrar lineas muy largas. 
    					(not filename.startswith('.') and
					filename.endswith(('.pyc', '.pyo')))
  • * A veces es mas apropiado usar un slash "/".
  • * Una nueva linea despues de un operador. 
    					resultado = variable_con_nombre_muy_largo_1 + 
								variable_con_nombre_muy_largo_2
  • * Una linea de espacio entre funciones, dos entre clases.
  • * Colocar un espacio despues de las comas. 
    					numeros = [1, 2, 3, 4, 5]
  • * Usar un espacio despues de ":" en los diccionarios, no antes. 
    					dic = {"llave1": valor1, "llave2": valor2}
  • * Usar espacios alrededor de asignaciones y comparaciones.
  • * Una linea de espacio entre funciones, dos entre clases.
  • * No colocar espacios extra dentro de los parentesis ni en las listas de argumentos.
  • * Usar nombres descriptivos, evitar nombres confusos.
  • * Usar underscores (pisos) evita confusiones con palabras claves o modulos del sistema.
  • * Usar underscore al inicio de una variable indica uso privado.

    		# MEJOR PERO MALO IGUAL
